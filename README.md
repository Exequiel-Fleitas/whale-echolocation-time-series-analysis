# Whale Echolocation Time Series Analysis

This project analyzes whale echolocation signals using time series and signal processing techniques in R.

## Methods
- ACF/PACF Analysis
- Ljung-Box Testing
- Spectral Density Estimation
- Butterworth Filtering

## Tools
- R
- tswge
- astsa
- ggplot2

## Key Findings
- Significant temporal dependence detected.
- Dominant frequency components identified.
- Low-frequency structure captured through filtering.
